# 🌦️ Weather App

A modern, responsive **Weather Forecasting Web Application** built using **HTML, CSS, JavaScript, and React.js**.  
It allows users to fetch **real-time weather information** for any city worldwide using the **OpenWeatherMap API**.

---

## 🚀 Features

- ✔️ **Real-time Weather Data** (Temperature, Humidity, Wind Speed, etc.)
- ✔️ **Search Any City Worldwide**
- ✔️ **Clean & Modern UI**
- ✔️ **Responsive Design** (Mobile-friendly)
- ✔️ **Dynamic Weather Icons**
- ✔️ **Error Handling for Invalid Cities**
- ✔️ **Fast API Integration using Fetch/Axios**

---

## 🛠️ Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **OpenWeatherMap API**
